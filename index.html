Optimizing Deep Learning Integration in ADAS For Enhanced Vehicle Safety

This research explores the implementation of YOLOv5 models within Advanced Driver Assistance Systems (ADAS) to enhance vehicle safety through improved object detection. The study evaluates various YOLOv5 models, focusing on their accuracy and speed in vehicle detection, demonstrating that the YOLOv5s model, in particular, excels in both aspects. The findings underscore the importance of high-quality and diverse training data and suggest that YOLOv5 models can significantly advance ADAS functionalities, contributing to safer smart city transportation systems.

Fraud Transaction Detection using Python

This project involves developing a comprehensive system to process and analyze transaction data, focusing on detecting fraudulent activities. It comprises four main modules: the dataset module efficiently reads and organizes transaction data, the distance module calculates distances between transactions, the statistics module computes various statistical measures and identifies anomalies, and the test module provides a user-friendly interface for interaction. The implementation emphasizes clean, readable code and enhances skills in handling large datasets, statistical analysis, and Python programming.
